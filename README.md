Работа выполнена Руденко Фёдором 
Курсовая работа по реализации программной системы для функционирования аэропорта.
Реализованы три пользователя (Администратор, Менеджер и Пилот)
Возможности администратора:

  - Добавление и удаление пользователей
  - Добавление и удаление самолётов
  - Ввод координат взлётно-посадочной полосы

Возможности менеджера:

  - Рассмотрение заявок на аренду самолётов
  - Расчёт стомости завершённых аренд
Возможности пилота:

  - Подача запросов на аренду самолёта
